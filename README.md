
# 🚀 REST_API

A simple and minimal **Node.js + Express** based REST API that handles basic HTTP POST requests. Designed as a starter backend template for learning, prototyping, or integrating with frontends.

---

## 📦 Tech Stack

- **Node.js**
- **Express.js**
- **Nodemon** (for auto-reloading in development)
- **JSON parsing** via built-in middleware

---


---

## ⚙️ Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:kauntiaakash2/REST_API.git
   cd REST_API
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. (Optional) Run with nodemon:
   ```bash
   npx nodemon index.js
   ```


---

## 📌 Scripts

```bash
npm start     # Run using Node.js
npm run dev   # Run using Nodemon (dev mode)
```
